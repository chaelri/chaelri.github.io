Overview

How a tap travels

One BOOT button. One Firebase round-trip. The same servo (for the autoclicker) or IR LED / servo (for the aircon) that responds to the existing phone remotes sees an identical payload — the pocket remote just speaks for you.

1. Tap or hold
Tap fires the current mode. Hold flips the mode and saves it to NVS.
2. HTTPS PUT
One REST write to the right Firebase path — same shape the phone remotes use.
3. Target reacts
Autoclicker's SSE stream / aircon's poller fire within ~150–1000 ms.
Hardware

Three boards + a battery

Stacked from top to bottom: ESP32-C3 + 0.42" OLED dev board, TP4056 USB-C charger/protection, and the LiPo cell. All three drawn top-down as hand-drawn SVG — no photos, no image files.

ESP32-C3 + 0.42" OLED
01space-style dev board. SSD1306 wired internally on GPIO5/6. BOOT on GPIO9.
TP4056 + DW01
USB-C charge in, protected 3.0–4.2 V out. Use-while-charging works.
LiPo 102050 · 1000 mAh
~12–24 h with WiFi modem sleep. Charges to 4.2 V via TP4056.
Wiring

Four connections in total

Four wires for the power chain, plus the USB-C charge cable. No resistors, no MT3608 — the ESP32's onboard 3.3 V LDO handles the battery voltage range directly. The OLED itself is the low-battery indicator: when it starts to flicker around ~3.5 V, plug the TP4056 in.

# From To Wire / part Notes
priority_high Before plugging the ESP32 in

The TP4056 output sits at whatever the LiPo is at (3.0–4.2 V). That is safe to put on the ESP32's 5V pin — the onboard LDO drops it to 3.3 V with ~250 mV of headroom. Do not connect it to the V3 pin; 4.2 V on the regulated 3.3 V rail exceeds the chip's spec.

// Active-LOW on GPIO9 (BOOT button). Tap if released before TAP_MAX_MS
// and no hold has fired. Hold fires the moment press duration crosses
// HOLD_MS — user gets immediate feedback rather than waiting for release.
void readButton() {
  int level = digitalRead(BTN_PIN);
  unsigned long now = millis();

  if (level != btnLastLevel) {
    btnLastLevel = level;
    btnLastEdge  = now;
    return;
  }
  if (now - btnLastEdge < DEBOUNCE_MS) return;

  bool nowPressed = (level == LOW);
  if (nowPressed != btnPressed) {
    btnPressed = nowPressed;
    if (btnPressed) {
      btnDownAt = now;
      holdFired = false;
    } else {
      unsigned long heldFor = now - btnDownAt;
      if (!holdFired && heldFor <= TAP_MAX_MS) doTap();
    }
  } else if (btnPressed && !holdFired && (now - btnDownAt >= HOLD_MS)) {
    holdFired = true;
    doHold();
  }
}

bool putJson(const char* url, const char* body) {
  if (WiFi.status() != WL_CONNECTED) return false;
  HTTPClient http;
  if (!http.begin(url)) return false;
  http.addHeader("Content-Type", "application/json");
  int code = http.PUT(body);
  http.end();
  return (code >= 200 && code < 300);
}
